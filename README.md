# Estrutura de Dados I

Repositório com exercícios e aulas práticas da disciplina **Estrutura de Dados I** — UTFPR Campus Ponta Grossa, Bacharelado em Ciência da Computação.

## Conteúdo

| Pasta | Descrição |
|-------|-----------|
| `17-03/` | Aula 01 e Lista de Exercícios nº 1 – Strings |

### Arquivos

- `aula01.c` — Experimentos com strings: leitura, comparação, arrays de strings e geração aleatória de nomes.
- `exercicio_string.c` — Lista de exercícios sobre manipulação de strings em C (34 exercícios).

## Compilação

```bash
gcc aula01.c -o aula01
gcc exercicio_string.c -o exercicio_string
```

## Tecnologias

- Linguagem C
- GCC
